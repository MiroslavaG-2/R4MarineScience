## MB5370 - R for Marine Science; Advanced data wrangling

This repository contains the code, outputs, and supporting materials developed throughout _MB5370 - R for Marine Science_. The module aims to develop and build upon data wrangling skils to apply to ecological data and allow the testing of hypoteses. 

This repository is organised by workshop, with each workshop further developing the knowledge and data wrangling skills. The primary aim of the code contained within this repository is educational, and provides examples of data handling, debugging, and wrangling. As well as a real dataset which can be used to practice the techniques learned in earlier workhops. 

Workshop 1 introduces aspects of data wrangling by having an AI-off phase to properly pay attention and understand the code and what it does. Workshop 2.1 was aimed at doing more advanced data wrangling, and common issues encountered. It covers aspects like wrangling names and dates, separating and uniting data tables, and handling missing values. Workshop 2.2 is aimed to put our data wrangling knowledge to the test and use real ecological data, wrangle it using the skills learned in workshop 2.1 and use the data to test some hypotheses. The data used for this last workshop contained information on four zones and four predatory fish species along the Ross River Estuary, collected over a 30-day period.

Finally the datasets used throughout the workshops are sourced from publicly available R packages. Additionally, some of the datasets provided in this repository were also used to demonstrate concepts throughout the workshops, to better understand data wrangling techniques and to gain more familiarity working with it.
